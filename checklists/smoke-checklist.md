# Checklist de Smoke Test

Este checklist visa garantir que as principais funcionalidades críticas do sistema (caminho feliz) estão funcionando após um novo deploy, antes de testes mais profundos.

- [ ] A página inicial (Login) carrega corretamente
- [ ] É possível realizar login com sucesso usando usuário válido
- [ ] Os produtos da vitrine são exibidos com imagens, títulos e preços
- [ ] É possível adicionar ao menos um produto ao carrinho
- [ ] O contador no ícone do carrinho é atualizado ao adicionar produtos
- [ ] É possível acessar a página do carrinho e visualizar o produto
- [ ] É possível completar o fluxo de checkout inteiro e ver a tela de confirmação ("Thank you for your order")
- [ ] É possível fazer o logout e retornar à tela de login
