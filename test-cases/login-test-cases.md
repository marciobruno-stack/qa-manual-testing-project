# Casos de Teste – Login

| ID | Caso de Teste | Passos | Resultado Esperado |
| :--- | :--- | :--- | :--- |
| CT-001 | Login válido | 1. Abrir login <br> 2. Inserir email válido <br> 3. Inserir senha válida <br> 4. Clicar Login | Usuário autenticado |
| CT-002 | Senha inválida | Inserir email válido e senha inválida | Mensagem de erro |
| CT-003 | Campos vazios | Clicar Login sem preencher | Mensagem “Campos obrigatórios” |
| CT-004 | Email inválido | Inserir email sem formato válido | Mensagem “Email inválido” |
