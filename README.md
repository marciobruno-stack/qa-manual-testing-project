# 📌 Projeto de Testes Manuais – QA

Portfólio criado para demonstrar habilidades práticas em testes manuais, incluindo plano de testes, casos de teste, cenários, checklists, relatórios de bugs e evidências.

## 🎯 Objetivo

Mostrar competências essenciais de QA:

*   Criação de casos de teste
*   Testes funcionais
*   Testes exploratórios
*   Testes de API (opcional)
*   Documentação de bugs
*   Regressão e smoke tests
*   Organização e clareza

## 🛠 Ferramentas utilizadas

*   Markdown
*   Google Sheets / Excel
*   Navegador (Chrome)
*   Site de testes: saucedemo.com
*   Prints de evidências

## 📂 Estrutura do projeto

*   `/test-plan` → plano de testes
*   `/test-cases` → casos de teste detalhados
*   `/test-scenarios` → cenários de teste
*   `/bug-reports` → relatórios de bugs
*   `/evidences` → prints de execução
*   `/checklists` → checklists de regressão e smoke

## 📈 Resultados

Este projeto demonstra capacidade de:

*   Documentar testes com clareza
*   Identificar falhas
*   Criar cenários completos
*   Organizar evidências
*   Trabalhar com fluxo de QA real
