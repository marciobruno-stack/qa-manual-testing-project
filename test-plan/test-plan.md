# Plano de Testes – Sistema de Login

## Objetivo
Validar o funcionamento do fluxo de login garantindo que usuários válidos consigam acessar o sistema e que erros sejam exibidos corretamente.

## Escopo
*   Login com credenciais válidas
*   Login com credenciais inválidas
*   Login com campos vazios
*   Comportamento de mensagens de erro
*   Redirecionamento pós-login

## Riscos
*   Falha no backend
*   Erros de autenticação
*   Mensagens incorretas
*   Falha no redirecionamento

## Critérios de entrada
*   Ambiente disponível
*   Usuário de teste criado

## Critérios de saída
*   Todos os casos executados
*   Bugs documentados
